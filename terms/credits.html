Utilisation de l'IA

Dans une volonté de transparence sur l'utilisation de l'IA sur notre projet Flucilien, l'utilisation de l'IA est listé ici par chaque participant au projet.

    • Utilisation de ChatGPT 5 pour la relecture et la correction grammaticale des textes rédigés.

      Pas de création de code de zéro, aide pour débuggage principalement.
  • Tom Delavigne
    • Utilisation de ChatGPT 5 pour une aide au déboggage sur le code traitant les données de fréquientation sur la manière de procéder
    • Utilisation de GitHub Copilot pour un accompagnement sur le filtrage des données de fréquentation des gares ainsi que pour du déboggage en général.
    • Utilisation de GitHub Copilot pour un accompagnement lors la création d'un « loader » pour un gain de temps.
